🎯 Best Way to Deploy Your Movie Recommendation System

Since your project uses Django (backend), the easiest option is:

👉 Deploy on Render (FREE & beginner-friendly)

🚀 METHOD: Deploy using Render (Step-by-Step)
🔹 Step 1: Prepare your project

Make sure these files exist in your project:

✅ 1. requirements.txt

If not, create:

pip freeze > requirements.txt
✅ 2. Procfile (very important)

Create a file named Procfile (no extension)

web: gunicorn movie_recommendation.wsgi
✅ 3. Install Gunicorn
pip install gunicorn

Then update requirements:

pip freeze > requirements.txt
✅ 4. Update settings.py

Add:

ALLOWED_HOSTS = ['*']
✅ 5. Static files fix

Add in settings.py:

STATIC_ROOT = os.path.join(BASE_DIR, 'staticfiles')
🌐 Step 2: Deploy on Render
🔹 Go to:

👉 https://render.com

🔹 Steps:
Sign up using GitHub
Click New → Web Service
Connect your GitHub repo
👉 GitHub
Select your project repo
🔹 Fill details:
Name: movie-recommendation
Environment: Python 3
Build Command:
pip install -r requirements.txt
Start Command:
gunicorn movie_recommendation.wsgi
🔹 Click Deploy

⏳ Wait 5–10 minutes…

🎉 DONE!

You’ll get a live link like:

https://movie-recommendation.onrender.com
⚠️ Common Errors (Important)
❌ Static files not loading

Fix:

python manage.py collectstatic
❌ Module not found

Fix:

Ensure all libraries in requirements.txt
❌ Large model/dataset issue

👉 Don’t upload large files
Use:

Google Drive
Load dynamically
